# Animal-classification-using-pre-trained-model
A deep learning-based image classification project that identifies different animal species using a pre-trained convolutional neural network (CNN). This project leverages transfer learning to achieve high accuracy with minimal training time.

***Overview***

>>This project uses a pre-trained model (like MobileNet / VGG / ResNet) to classify animal images into different categories.

Instead of training from scratch, we:

--Use a model trained on large datasets (like ImageNet)
--Fine-tune it for animal classification

***Features***

>>Uses transfer learning for faster training
  --Classifies multiple animal categories
  --High accuracy with less data
  --Supports custom image prediction
  --Training & validation visualization

>>Tech Stack
  --Python 🐍
  --TensorFlow / Keras
  --NumPy 
  --Pre - trained model(MobileNetV2)
